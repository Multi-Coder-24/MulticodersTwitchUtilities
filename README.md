# Multicoder Twitch Utilities (MTU)

Welcome to the new Multicoder Twitch Utilities Application For Twitch Streamers. This application is a combination of a couple of my applications JTPoll and Multicoder's Twitch Giveaway Utility program.

One of the new key features is with how your Twitch Auth Token get's used, The application will no longer save your token for security purposes.

## Features

* Admin configuration Panel
* Poll feature with Presets support
* Giveaway feature
* Moderation tool for Poll and Giveaway features

See [Moderation] (https://multi-coder-24.github.io/MulticodersTwitchUtilities/Moderation)
